Meine persönliche Entwicklerseite soll zum einen auf mich aufmerksam machen, zum anderen dient Sie mir als Learning-Site um neue Komponenten und Features aus dem Angular Framework zu testen. So wurden verschiedene Komponenten aus den Angular Materials genutzt und ein responsive Design implementiert.

Verwendete Technologien

Angular 17 & CDK Materials

Diese Single Page Application (SPA) ist im modernen Angular Framework V17 entwickelt. Das Webdesign ist mit Angular Materials Komponenten umgesetzt.

TypeScript

Das Angular Framework basiert auf der TypeScript Programmiersprache - eine Erweiterung von JavaScript - und ermöglicht die objektorientierte Programmierung im Frontend. TS wird genutzt um die Funktionalität der Komponenten zu implementieren.

SCSS und Sass

SCSS dient als CSS-Präprozessor und vereinfacht die Nutzung, z.B. durch den Einsatz von Variablen und Schleifen. Außerdem verwendet FlexBox CSS einige Sass mixins um somit die Kompatibilität unterschiedlicher Browser zu garantieren.
